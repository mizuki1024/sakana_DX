# 運用マニュアル

漁市場デジタル台帳システムの日常操作マニュアルです。

## ログイン

1. ブラウザでシステムURLにアクセス
2. メールアドレスとパスワードを入力
3. 「ログイン」ボタンをクリック

ログインできない場合は管理者に連絡してください。

## 日常業務の流れ

```
朝: 水揚げ登録
 ↓
日中: 売上登録
 ↓
月末: 請求書作成・送付
 ↓
入金確認後: 入金済み処理
```

---

## 水揚げ登録

### 新規登録

1. サイドバー「水揚げ一覧」をクリック
2. 「新規登録」ボタンをクリック
3. 以下を入力:
   - **日付**: 水揚げ日（デフォルトは今日）
   - **仕入先**: 漁船・漁業者を選択
   - **魚種**: 魚の種類を入力
   - **ロット番号**: 管理用の番号（重複不可）
   - **重量**: kg単位で入力
   - **等級**: A/B/Cなど（任意）
   - **保管場所**: 冷蔵庫番号など（任意）
4. 「登録」ボタンをクリック

### ステータスの意味

| ステータス | 意味 |
|-----------|------|
| LANDED | 水揚げ済み（未販売） |
| SOLD | 販売済み |
| INVOICED | 請求済み |

---

## 売上登録

### 新規登録

1. サイドバー「売上一覧」をクリック
2. 「新規登録」ボタンをクリック
3. 以下を入力:
   - **水揚げ**: 未販売の水揚げから選択
   - **買受人**: 販売先を選択
   - **売上金額**: 円単位で入力
4. 「登録」ボタンをクリック

登録すると、対応する水揚げのステータスが「SOLD」に変わります。

### 仕切書の確認

1. 売上一覧から対象の売上をクリック
2. 売上詳細画面で「仕切書を表示」をクリック
3. 印刷する場合は「印刷」ボタンをクリック

---

## 請求書作成

### 新規作成

1. サイドバー「請求書一覧」をクリック
2. 「新規作成」ボタンをクリック
3. 以下を入力:
   - **買受人**: 請求先を選択
   - **対象月**: YYYY-MM形式で入力（例: 2026-06）
4. プレビューで集計金額と売上件数を確認
5. 「作成」ボタンをクリック

作成時に、対象月の未請求売上が自動的に集計されます。

### ステータス管理

| ステータス | 意味 | 次のアクション |
|-----------|------|----------------|
| DRAFT | 下書き | 「送付済みにする」 |
| SENT | 送付済み | 「入金済みにする」 |
| PAID | 入金済み | （完了） |

### 請求書の印刷

1. 請求書一覧から対象の請求書をクリック
2. 詳細画面で「印刷」ボタンをクリック
3. ブラウザの印刷ダイアログが開く
4. PDFとして保存または印刷

## マスタ管理

### 買受人の登録

1. サイドバー「マスタ管理」→「買受人一覧」をクリック
2. 「新規登録」ボタンをクリック
3. 以下を入力:
   - **名前**: 会社名・店舗名
   - **コード**: 管理用コード（重複不可）
   - **メール**: 連絡先メール（任意）
   - **電話番号**: 連絡先電話（任意）
4. 「登録」ボタンをクリック

### 仕入先の登録

1. サイドバー「マスタ管理」→「仕入先一覧」をクリック
2. 「新規登録」ボタンをクリック
3. 以下を入力:
   - **名前**: 漁船名・漁業者名
   - **コード**: 管理用コード（重複不可）
   - **連絡先**: 電話番号など（任意）
4. 「登録」ボタンをクリック

### 削除について

- 削除は**管理者のみ**実行可能
- 売上データがある買受人は削除できません
- 水揚げデータがある仕入先は削除できません

## CSVエクスポート

各一覧画面で「CSVエクスポート」ボタンをクリックすると、データをCSVファイルでダウンロードできます。

- 水揚げ一覧: 全水揚げデータ
- 売上一覧: 全売上データ
- 請求書一覧: 全請求書データ

CSVファイルはExcelで開けます。

## 権限について

| 権限 | できること |
|------|-----------|
| ADMIN（管理者） | 全操作、削除、監査ログ閲覧 |
| STAFF（スタッフ） | 登録・編集・閲覧（削除不可） |

## トラブルシューティング

### ログインできない

- メールアドレスとパスワードを確認
- Caps Lockがオフになっているか確認
- 管理者にアカウント状態を確認

### 登録ボタンが押せない

- 必須項目（*マーク）がすべて入力されているか確認
- エラーメッセージを確認

### 「このコードは既に使用されています」エラー

- ロット番号やコードが重複しています
- 別の番号を入力してください

### 「削除には管理者権限が必要です」エラー

- 削除は管理者のみ実行可能です
- 管理者に依頼してください

### データが表示されない

- ブラウザを再読み込み（F5キー）
- ログアウトして再ログイン
- 改善しない場合は管理者に連絡
